#  SMARTSPEND

SmartSpend is a web-based financial management platform designed to help parents monitor their child's daily expenses, set spending limits, and encourage responsible money management.

---

##  Live Website

:- https://sites.google.com/view/smartspend2

---

##  About the Project

SmartSpend helps parents:

- Track their child's daily expenses
- Set monthly spending limits
- Receive reminders when the spending limit is close to being reached
- Build healthy financial habits in children

The platform is simple, user-friendly, and completely free to use.

---

##  Features

-  Daily expense tracking
-  Monthly spending limits
-  Spending limit alerts
-  Percentage-based budget tracker
-  Date & Time recording
-  Google Sheets integration
-  Google Forms for expense entry
-  Parent-friendly dashboard
-  Ad-free interface
-  Government schemes & benefits section

---

##  Technologies Used

- Google Sites
- Google Forms
- Google Sheets
- HTML Embedding

---

## 🎯 Objective

To promote financial responsibility among children while providing parents with an easy way to monitor and manage daily expenses.
